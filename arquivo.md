# Tutorial de linguagem markdown.

Aqui será documentado os principais tipos de formatações utilizadas no markdown tais como:


- Principais formatações:
  - Títulos e subtítulos  
  - negrito:
  - italico:
  - inserir links.

## Títulos e subtitulos

para criar títulos utiliza-se o jogo da velha/cerquila (#) os títulos e subtítulos vão do nível 1 ao nível 6.
# Título nível 1
## Título nível 2
### Título nível 3
#### Título nível 4
##### Título nível 5
###### Título nível 6

## Subtítulo

**Negrito:** Para usar **negrito** é utilizado dois asteriscos (*) dois antes e dois depois do texto selecionado.

**Links:** para inserir um link temos que digitar a palavra que desejamos que se torne o link entre colchetes ([Clique aqui!]) em seguida entre parenteses incluir o link (link aqui) tudo junto. [Clique Aqui](https://google.com.br)

**Listas:** Para criar lista não numeradas utilizamos o sinal de menos (-) seguido da lista, exemploa abaixo:
- Item um;
- Item dois;
- Item três.

**Citação:** para incluir um citação usamos o simbolo de maior (>) seguido de um espaço o que deseja que seja dito (cada linha deve possuir a formatação).

**Exemplo:**
> Não sei,
> Só sei que foi assim.


**Tabelas:** para criar tabelas usamos o (|) antes e depois do que iremos usar seguido do nome e fechando com o mesmo simbolo |Título1 | Título 2|
embaixo é necessário possuir | ------ | ------ | se não, não funcionará, os traços são usados para separar somente o título, não precisa usar nos abaixo ficando |nome|sobrenome|:. exemplo a abaixo:

|Nome|Sobrenome|
| ------ | ------ |
|Robson|Freitas|
|Atais|Ramos|
|Julia|Ramos|
